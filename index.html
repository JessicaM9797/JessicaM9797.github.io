费曼学习方法引出的思考

2018-03-04

进入大学后，不仅要学许多专业的知识，还要处理日常的人际关系，这一度让我感到力不从心，也因此对大脑的决策和思维模式产生了兴趣，这真的是非常神奇的一件事情，大脑就拳头大小的一块儿灰质，是如何影响着我们的一举一动，如何催生出复杂的情感和思绪。后来接触了正念，发现这是一个极好的解压方式，现在看来所谓压力也只是因为外界刺激产生的一种消极情绪反应，而实际上客观世界发生的事情和大脑里构想出的种种并不完全一样，可是在深陷压力的泥潭时，就会很难看到未来的希望，只是自顾自的越陷越深，因此我也发现积极的思考模式会极大的改善我们的生活。

你好哇

2018-02-10

你好哇这个神奇的博客，第一篇记录就留给使用你的过程吧

  • 用cmd或git bash打开work盘中的中的blog文件夹，用hexo new “xxx”来新建一篇博客
  • 用sublime编辑好之后保存
  • 还是在cmd或者git bash中输入以下命令（打开blog文件夹）
    hexo clean
    hexo generate
    hexo server -p 5003
    hexo deploy
