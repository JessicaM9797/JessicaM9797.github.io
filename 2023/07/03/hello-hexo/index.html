hexo博客搭建(Github部署)

1. Github相关配置

  • github主页右上角选择New Reporsitory，仓库名要和帐号名称一致，如 yourusername.github.io

  • 安装git并配置邮箱和用户名

2. 安装node.js

安装完成后通过-v查看安装版本

3. 安装hexo

sudo npm install hexo-cli -g
hexo -v

4. 创建博客文件夹

在合适的目录创建存储博客的文件夹

通过hexo进行初始化

到这里就可以通过本地链接预览博客了

5. 为githubh帐号生成ssh密钥

退回到根目录下生成ssh密钥

点击github上右上角头像，选择左侧选项中的SSH and GPG keys，将id_rsa.pub里面的所有内容添加到SSH keys里的Key，Title随便填，然后A点击dd SSH key
